# Pure Helper

Pure Helper is a RuneLite external plugin focused on pure-account safety:

- combat style safety warnings
- quest and achievement diary risk checks against protected skills/caps
- local build preset import/export
- optional sidebar panel with safeguards and risk log

## Plugin Usage

After installing and enabling Pure Helper in RuneLite:

1. Open the Pure Helper sidebar panel.
2. In `Skills`, choose your build preset, select protected skills, and set optional max protected levels.
3. In `Safeguards`, configure quest and diary blocking behavior.
4. In `Combat`, configure warning infobox behavior and unsafe style visibility.
5. In `Risk Log`, review recent risky style selections.
6. In plugin settings, optionally customize `Accent color` and related visual preferences.

## Tips

- If you are testing a new account build, start by protecting all relevant XP types first, then tune caps.
- Keep `Risk Log` open while training or questing to quickly spot risky interactions.
