Custom tab icons for Pure Helper sidebar tabs.

Drop your PNG files in this folder using these exact names:

- `safeguards.png` -> Safeguards tab
- `combat.png` -> Combat tab
- `skills.png` -> Skills tab
- `log.png` -> Risk Log tab

Recommended:

- Size: `30x30` px (or square, it will be resized to fit)
- Background: transparent PNG
- Style: high-contrast for dark UI

If a file is missing, the tab will show a gray placeholder icon.
